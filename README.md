# Lernapps

Eine Sammlung interaktiver Werkzeuge für den Mathematikunterricht und statistische Analysen. Die Anwendungen sind darauf ausgelegt, mathematische Konzepte wie Funktionen, Brüche und Statistiken visuell darzustellen und berechenbar zu machen.

## Werkzeuge im Detail

### Mathematik-Tools (HTML)

Diese Tools können direkt im Browser geöffnet werden und bieten interaktive Eingabemöglichkeiten:

* **Bruchrechner (`Bruchrechner.html`)**: Ein Tool zur Durchführung und Visualisierung von Grundrechenarten mit Brüchen.
* **Lineare Funktionen (`lineare-funktionen.html`)**: Visualisierung und Berechnung von Steigungen, y-Achsenabschnitten und Funktionsgraphen linearer Gleichungen.
* **Quadratische Funktionen (`quadratische_fkt.html`)**: Untersuchung von Parabeln, Scheitelpunkten und Nullstellen quadratischer Gleichungen.

### Statistik & Spiel (HTML)

* **Boxplot-Generator (`boxplots.html`)**: Ein Werkzeug zur Erstellung von Boxplots basierend auf eingegebenen bzw. vorgegebenen Datensätzen (Minimum, Quartile, Median, Maximum).
* **Bingo-Würfel-App (`bingo_wuerfel_app_ver1.1.html`)**: Eine digitale Unterstützung für Bingo-Spiele oder zur Zufallszahlengenerierung im Unterrichtskontext.

### Python-Anwendungen

* **PolyGeniusBeta (`PolyGeniusBeta.py`)**: Ein Python-basiertes Skript (Beta-Stadium), das sich auf die Arbeit mit Polynomen konzentriert (z. B. Termumformungen oder Berechnungen höherer Ordnung).

## Nutzung

### Web-Apps (HTML/JS)
Die HTML-Dateien sind eigenständig lauffähig. Es ist keine Installation von Frameworks oder Webservern notwendig:
1.  Repository klonen oder Datei herunterladen.
2.  Die gewünschte `.html`-Datei in einem modernen Webbrowser (Chrome, Firefox, Safari, Edge) öffnen.

### Python-Skript
Um `PolyGeniusBeta.py` zu nutzen, muss eine Python-Umgebung installiert sein:
